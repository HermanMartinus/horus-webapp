How it works

Step 1

Type in your master password. This is (preferably) a phrase that you never write down or tell anyone.

Step 2

Type in the name of the service you would like to access (eg: Facebook). Case does not matter. This is how you retrieve the service's password.

Step 3

A password is generated in the obscured field. The password is generated by doing a one way hash of the secret phrase combined with the service name and then decorated with special characters.

Step 4

Set the password of your service to the generated password. Now you never have to remember that password ever again. Every time you want to log in, just come regenerate it and copy and paste it again.
